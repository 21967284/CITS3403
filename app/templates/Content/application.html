Explanation

The application layer is the top-most layer in the TCP/IP Model. This is the layer that is responsible for communication between different computers (sometimes known as client and server).

The application layer uses several protocols (ways of communicating) to ensure that we can communicate using a variety of mediums (e.g. text, files, etc.) with each other.

In this layer, we'll explore what some of those protocols are

HyperText Transfer Protocol? Sounds like an intergalactic battleship communication protocol!

Unfortunately this HTTP protocol is not quiet as exciting as that.

But the HTTP protocol also has its own brilliance. It is very widely used in the internet, because it is a great way to communicate data in the world wide web.

HTTP is known as a request-response protocol. It functions just as the name suggests, a client computer can request information using HTTP and the server computer can respond to this HTTP request.

HTTP is made up of a header and a body. The client computer can put information in either the header or the body to request different kinds of data from the server.

FTP is similar to HTTP in that it is allows client computers and server computers to communicate with each other.

The only difference is the type of information they communicate with each other. As the name suggests, HTTP communicate text-like information and FTP communicates in files! FTP's speciality is transfering larger files.

FTP protocol allows files to be copied and distributed form one computer to another by establishing two connections. One of these connections is required for data transfer and the other one is required for control information.

The secure shell protocol is also a protocol to communicate between client and server. However, a key difference between this protocol and the other ones is that SSH provides a secure channel of communication.

In the SSH protocol, before computers can communicate with each other, the computers must authenticate themselves first. That is, they must prove they are who they say they are. Once they've done this, then they can communicate with other computers.
